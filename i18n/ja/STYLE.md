# 日本語翻訳スタイルガイド — HongKong.md

## 基本原則

> 翻訳ではなく、書き直し。日本語の記事は、最初から日本語で書かれたように読めるべきです。

## 用語表

| 中文 | 日本語 | 備考 |
|------|--------|------|
| 香港 | 香港（ホンコン） | 「ホンコン」で統一 |
| 廣東話 / 粵語 | 広東語 | 書き言葉の場合は「書面広東語」と明記 |
| 飲茶 | 飲茶（ヤムチャ） | 日本語でもそのまま通じる |
| 茶餐廳 | 茶餐廳（チャーチャンテン） | 初出時に「香港式カフェ」と補足 |
| 回歸 | 返還 | 1997年の出来事を指す場合は大文字相当 |
| 基本法 | 基本法 | そのまま使用可 |
| 立法會 | 立法会（レジコ） | 略称「レジコ」は初出時にフル表記後に使用 |
| 行政長官 | 行政長官 | 1997年以降は「総督」を使わない |
| 港鐵 | MTR | 補足不要、国際的に認知されている |

## 文化的コンテキスト

文化特有の概念を翻訳する際は、簡潔な説明を追加してください：

- **一國兩制** → 「一国二制度」（中国と香港の関係を規定する憲法的枠組み）
- **功夫** → 「カンフー」— 国際的に認知されている
- **粵劇** → 「広東オペラ」— ユネスコ無形文化遺産として簡潔に説明
- **HKD / 港幣** → 日本円での概算を併記：「100香港ドル（約1,900円）」

## トーンとスタイル

- 親しみやすいが権威のある文体 — 雑誌の特集記事のイメージ
- 学術用語を避け、専門用語は初出時に説明
- 日本の読者に馴染みのある比較や類例を活用
